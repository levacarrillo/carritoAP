# Software for carritoAP

### Compilation
```
 cd ~/carritoAP/catkin_ws
 catkin_make
 source ~/carritoAP/catkin_ws/devel/setup.bash
```

### Testing
```
 roslaunch surge_et_ambula carritoAP.launch 
 roslaunch surge_et_ambula carritoAP_simul.launch 
```
